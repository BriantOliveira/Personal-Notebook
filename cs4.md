## Class 1: Bit Manipulation

Bitwise Operations

And **&**

Or **\|**

XOR **^**

NOT** ~**

LEFT SHIFT **&lt;&lt;**

RIGHT SHIFT **&gt;&gt;**

**And** - Only true if both input bits are true

0 & 0 = 0

1 & 0 = 0

0 & 1 = 0

1 & 1 = 1

**XOr** - True if any input bit is true

0 \| 0 = 0

1 \| 0 = 1

0 \| 1 = 1

1 \| 1 = 1
